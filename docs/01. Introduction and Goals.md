Introduction and Goals
======================

## Vision 
```
"Enable direct communication between users and agents for an improved service satisfaction"
```
___

## Requirements Overview

### Why

- Improve user satisfaction by enabling them to receive quick solutions for their problems
- Reduce the effort to create a new ticket for a customer / gather all required information to solve a ticket

### How

- Reusing existing functionality to quickly create new processes
- Including in new POWA client

### What

- Integrate Messaging (former Smoope) into Processes

___
## Quality Goals

| Priority | Quality Goal    | Scenario                                                                                    |
| -------- | --------------- | ------------------------------------------------------------------------------------------- |
| 1        | Configurability | It should be easy to install and configure Messaging for/on customer environments           |
| 2        | Monitoring      | The integration should provide logging and monitoring mechanism based on the "Book of play" |
