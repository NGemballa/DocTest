Architecture Constraints
========================

 | Constraint                   | Explanation                                                                                                                                              |
 | ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
 | Hybrid operation             | While Processes is currently operated mostly on-premise Messaging will be hosted and operated in our cloud environment, which leads to a hybrid approach |
 | Minimal configuration effort | The Messaging integration should be usable out of the box with minimal configuration effort to have a quick adaption by our customers                    |
